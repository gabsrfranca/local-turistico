## 🚀 Sobre o Projeto

Este projeto foi criado com o objetivo de apresentar locais turísticos de forma atrativa e organizada. A página foca-se na experiência do utilizador, utilizando imagens de alta qualidade e textos descritivos para incentivar a exploração de novos destinos.

## ✨ Funcionalidades

- **Header:** Destaque inicial com imagem impactante e chamada para ação.
- **Galeria de Destinos:** Exibição de locais turísticos com imagens e descrições.
- **Footer:** Considerações finais para o leitor.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Semantic tags e text formating
- **CSS3:** Estilização e Flexbox.
- **Google Fonts:** Tipografia personalizada para melhorar a leitura.
